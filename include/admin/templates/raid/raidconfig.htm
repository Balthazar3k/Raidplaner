Einstellungen zu Verändern könnte schlimme Ausmase habe, einstellungen nur für Vortgeschrittene!
Vorhande einstellungen können auch Bearbeitet werden, aber sie Dürfen kein Zeile nach oben oder nach unter geschoben werden.
Hinter jeder einstellung befindet sich eine FESTE "id" die im ganzen Raidplaner alle werte richtig Darstellen!

Neue Zeilen Hinzufügen, sollte Jedoch ohne Problem Möglich sein.
Felder in den man keine Zuordnung erkennen kann aber durchaus eine 0 oder 1 haben bedeutet meist 0 = Aus 1 = An,
bsp. bei Klassen stehen diese zahlen, sie Steuern ob noch klassen gesucht werden oder nicht! 1=Wir Suchen noch 0=Wir Suchen keine dieser Klassen mehr.
